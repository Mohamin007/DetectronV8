DetectronV8

Real-time object detection powered by Django and YOLOv8, designed to upload an image, process it, and present clear visual detections with confidence scores.

Goal

  • Upload an image and run YOLOv8 detection with a single flow.
  • Render the processed image with labels and bounding boxes.
  • Provide readable detection output with confidence values.

How It Works

  1. User uploads an image from the browser.
  2. Django stores the file and triggers detection logic.
  3. YOLOv8 predicts classes and confidence for found objects.
  4. Processed image and detection details are returned to the page.

Tech Stack

  • Backend: Django
  • Model: YOLOv8 (Ultralytics)
  • Language: Python
  • Frontend: HTML and Django Templates
  • Image Processing: OpenCV
